Edge AI-Driven Incident Response SystemDeveloped by Team Goal Getters An intelligent, edge-native security hub that integrates multi-sensor data with real-time computer vision to automate incident detection and visual verification.⚠ The ProblemHuman Error: Manual monitoring is highly prone to errors.Response Delays: Reliance on manual checks creates dangerous delays during emergencies.Key Risks:Slow response to fires and hazardous gas leaks.Lack of real-time visual confirmation for remote operators.Absence of automated incident logging for forensic review.✔ The SolutionAutomated Security Hub: Combines multi-sensor data with live visual verification.High Performance: Achieves sub-second sensor-to-alert latency.Remote Oversight: Includes a web-based dashboard with an evidence log.🛠 System ArchitectureHardware: Edge ControllerThe system uses an Arduino as the local controller for real-time responsiveness:Sensor Suite:DHT11: Monitoring Temperature and Humidity.Gas/Flame Sensors: Detecting hazardous leaks and fire.Water Sensor: Monitoring for flooding and leaks.Local Alerts: Instantly triggers a physical buzzer and LED when thresholds are met.Communication: Packages data into JSON and sends POST requests via WiFi to the central server.Software: Backend & AIFlask Web Server: A Python-based "brain" that receives data, hosts the dashboard, and manages alert states.OpenCV Computer Vision:Manages a live camera feed for visual verification.Automatically captures timestamped snapshots during an alarm.Highlights the Region of Interest (ROI) with a red bounding box labeled CRITICAL during incidents.Evidence Logging: Saves snapshots to a /history folder, retaining the last 5 critical events.⚙ Operational SpecificationsAlarm Thresholds:Gas Level > 400 ppm.Temperature > 60 °C.Dashboard Features:Live telemetry for Temperature, Humidity, and Gas.Embedded live OpenCV video stream.Browser-based audio alarm using Web Audio API.Incident history view for the last 5 critical snapshots.🚀 Future ScopeImplementation of YOLO-based object and person detection.Integration of Cloud SMS and Email notifications.Scaling to a multi-node sensor network for large
